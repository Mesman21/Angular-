Розділ 1: Робота з фонами та границями

Прозорі границі.
Однією з хитрощів CSS є створення прозорих границь, коли фон елемента проглядається через рамку. Це досягається за допомогою властивості border-image, але простіше використовувати background-clip: padding-box;. Це обмежує застосування фону тільки до області заповнення (padding), залишаючи границю прозорою.

Багатошарові фони.
Замість використання декількох вкладених div для створення складних фонів, CSS3 дозволяє використовувати кілька фонових зображень на одному елементі, розділених комою. Це значно зменшує "засмічення" DOM. Наприклад: background: url(pattern1.png), url(pattern2.png);.

Розділ 2: Елегантна типографія

Ефект "обведення тексту" (Text Stroking).
Хоча SVG надає кращий контроль, CSS дозволяє створити простий ефект обведення за допомогою text-shadow. Використовуючи множинні тіні, можна імітувати обведення: text-shadow: 1px 1px #000, -1px -1px #000, 1px -1px #000, -1px 1px #000;. Це забезпечує чистий вигляд без необхідності дублювати елементи.

Фіксований фон для тексту (Text Clipping).
Ви можете змусити фонове зображення або градієнт "виглядати" через текст, використовуючи властивості background-clip: text та text-fill-color: transparent. Це створює вражаючі заголовки.

Розділ 3: Управління макетом

"Sticky Footer" (Прилиплий підвал).
Це класична проблема веб-дизайну: як зробити, щоб підвал завжди був внизу екрана, навіть якщо контенту мало. Найкраще рішення сьогодні — це використання Flexbox або Grid на елементі body/main, встановлюючи min-height: 100vh для контейнера та margin-top: auto для підвалу.

Створення складних фігур (Трикутники, Діаманти).
Трикутники створюються за допомогою властивостей border. Якщо встановити ширину і висоту елемента в 0, а три границі зробити прозорими, то залишається тільки один видимий трикутник. Це дозволяє створювати спливаючі підказки та стрілки без використання зображень.

Розділ 4: Адаптивні хитрощі

Співвідношення сторін для контейнерів.
Як створити адаптивний контейнер (наприклад, для відео), який зберігає співвідношення сторін (16:9) незалежно від ширини екрана? Використовуйте padding-top або padding-bottom у відсотках. padding-top: 56.25% (що дорівнює 9/16) забезпечить ідеальне співвідношення 16:9.
