Частина I: Анатомія веб-сторінки та HTML5

Розділ 1: З чого складається веб.
Інтернет — це сукупність HTML-документів, з'єднаних гіперпосиланнями. HTML (HyperText Markup Language) відповідає за структуру та контент, тоді як CSS (Cascading Style Sheets) відповідає за візуальне оформлення. Взаємодія забезпечується JavaScript.

Розділ 2: Семантика HTML5.
HTML5 представив важливі нові теги для семантичної розмітки, які допомагають пошуковим системам та допоміжним технологіям зрозуміти структуру документа. Основні семантичні теги: <header>, <nav>, <main>, <article>, <section>, <footer>. Використання цих тегів замість універсальних <div> робить код більш доступним та зрозумілим.

Розділ 3: Основи роботи з текстом та посиланнями.
Текст форматується за допомогою тегів заголовків (<h1> - <h6>), параграфів (<p>) та списків (<ul>, <ol>). Посилання (<a>) є основою гіпертексту, що дозволяє створювати зв'язки між документами. Важливо правильно використовувати атрибути href та target.

Частина II: Стилізація за допомогою CSS

Розділ 4: Каскад, Спадкування та Специфічність.
CSS працює за принципом каскаду: браузер вирішує, яке правило застосувати, ґрунтуючись на трьох факторах: спадкуванні (наприклад, колір тексту може успадковуватися від body), специфічності (наскільки точним є селектор), та послідовності (правило, що йде пізніше, має пріоритет). Розуміння цих принципів є ключем до уникнення непередбачуваної поведінки стилів.

Розділ 5: Модель блоку (Box Model).
Кожен HTML-елемент у CSS розглядається як прямокутний блок. Модель блоку складається з чотирьох частин, від зовнішньої до внутрішньої: Margin (зовнішній відступ), Border (рамка), Padding (внутрішній відступ) та Content (вміст). box-sizing: border-box — це сучасний і кращий спосіб розрахунку розмірів блоку, який включає padding та border у встановлену ширину елемента.

Розділ 6: Основи макетування.
Ранні методи макетування базувалися на float та position: absolute. Сучасний веб-дизайн майже повністю перейшов на Flexbox (для односпрямованих елементів, наприклад, навігації) та CSS Grid (для двовимірних макетів сторінки). Ці інструменти дозволяють створювати повністю адаптивні та гнучкі сітки з мінімальною кількістю коду.

Частина III: Візуальний дизайн та адаптивність

Розділ 7: Робота з кольором та зображеннями.
Колір можна задавати за допомогою ключових слів, HEX-кодів, RGB/RGBA або HSL. Зображення повинні бути оптимізовані за розміром. Для адаптивності слід використовувати max-width: 100% на зображеннях, щоб вони не виходили за межі своїх контейнерів.

Розділ 8: Адаптивний дизайн.
Адаптивний дизайн (Responsive Design) забезпечує коректне відображення сайту на всіх пристроях (мобільних, планшетах, десктопах). Основний інструмент — це медіа-запити (@media screen and (max-width: 768px)), які дозволяють застосовувати різні стилі на різних розмірах екрана.
